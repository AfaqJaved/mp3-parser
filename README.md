## Description

Mp3 Decoder App Buit with Nestjs

## Installation

```bash
$ npm install
```

## Running the app
### Default Port is 3000

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
